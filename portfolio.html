PLTW 1.1.9 Project

In this project, we culminated all the knowledge we learned throughout the unit 1.1. Our project displays a scene that consists of multiple flowers, a bee, and a beehive. We created these components by finding images online and creating them into GIFs in order to import them into Python Turtle. The flower was created using a for loop in order to shorten the process of importing and positioning six different flowers. In order to get the bee to move we used a while loop that will repeat six times. This allows the bee to fly from the hive, down to all the flowers, and collect the honey to then bring back up to the hive.

PLTW 1.2.5 Project Game

The objective of the Snake Survival game is to eat as many apples as possible without dying. The more apples you eat, the larger the snake grows. You can die by either running into yourself or the wall.The snake is controlled by the user using the movement keys. A scoreboard is presented on the screen and it is the users task to try to beat the highscore.

PLTW 1.3.1 Project Game

The objective of our game, Riddled Treasure, is to cross both bridges and reach the treasure on the other side. In order to cross the bridges you must answer two riddles correctly. If you answer incorrectly, you will fall into the lava and die. If both riddles are answere correctly, you will be able to get to the treasure where you will be prompted with a screen that congratulates you for yout succces.
